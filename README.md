# 📦 RipPack Frontend: The Viral Meme Card Trading UI

This is the premium frontend for RipPack, a digital card opening and collection game. Built for speed, engagement, and conversion.

## ✨ Features
- **Premium Glassmorphic UI**: Deep blacks, neon glows, and sophisticated blur effects.
- **3D Pack Opening**: Immersive card reveal animations with 3D rotations.
- **Vault/Inventory**: Track progress with a high-end collection dashboard.
- **Optimized Shop**: Integrated conversion flow for MVP bundles.
- **Micro-Animations**: Smooth transitions using modern CSS techniques.

## 🚀 Tech Stack
- **Framework**: React 19 + Vite
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Custom Design System)
- **Routing**: React Router 7
- **Payments**: Stripe Component Prep

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/tizianoguidoni/tgc-frontend.git
   cd tgc-frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Ensure your backend is running at `http://localhost:8000` (handled via Vite proxy in `vite.config.js`).

4. **Launch Development Server**:
   ```bash
   npm run dev
   ```

## 📦 Components Overview
- `/src/components/Dashboard.jsx`: The hero entry point for pack opening.
- `/src/components/Shop.jsx`: High-conversion glassmorphic shop grid.
- `/src/components/Inventory.jsx`: The collection vault with progress tracking.
- `/src/components/PackOpening.jsx`: The 3D animation engine for card reveals.
